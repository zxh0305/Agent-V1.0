# 智能任务处理 Agent 平台 V3.0

## 项目简介

智能任务处理 Agent 平台是一个功能完整的智能 Agent 系统，专为学生科研和学习场景设计。V3.0 版本实现了真实的服务能力，接入了多种AI和API服务。

### 核心特点

- **自然语言交互**：支持用户使用自然语言描述任务
- **智能意图识别**：接入大模型API，智能识别用户意图
- **智能任务规划**：自动将复杂任务拆分为可执行的子步骤
- **真实服务能力**：接入真实API，提供翻译、问答、文献搜索等服务
- **工具调用引擎**：内置8个学生高频工具，支持灵活扩展
- **可视化界面**：基于 Streamlit 的友好 Web 界面
- **配置化管理**：统一配置文件，一键更改API密钥

## V3.0 新增功能

### 真实服务能力

- **🤖 大模型接入**：支持 OpenAI、通义千问、智谱AI、Ollama 等大模型
- **🌐 真实翻译**：接入百度翻译、有道翻译API
- **📚 学术搜索**：接入 Semantic Scholar、arXiv、CrossRef 学术数据库
- **💬 智能问答**：基于大模型的智能问答能力
- **📝 智能摘要**：基于大模型的文本摘要生成

### 配置管理

- **统一配置文件**：所有API配置集中在 `settings.yaml`
- **热更新支持**：配置修改后可动态加载
- **配置验证**：自动验证配置有效性

## 快速开始

### 环境要求

- Python 3.9 或更高版本
- pip 包管理器

### 安装步骤

1. **克隆项目**
```bash
git clone <repository_url>
cd Agent-V1.0
```

2. **创建虚拟环境（推荐）**
```bash
python -m venv venv
# Windows
venv\Scripts\activate
# Linux/Mac
source venv/bin/activate
```

3. **安装依赖**
```bash
pip install -r requirements.txt
```

4. **配置API密钥**

编辑 `settings.yaml` 文件，配置所需的API密钥：
```yaml
llm:
  provider: "qwen"
  qwen:
    api_key: "your-api-key-here"

translate:
  provider: "baidu"
  baidu:
    app_id: "your-app-id"
    secret_key: "your-secret-key"
```

详细配置说明请参考 [CONFIG.md](CONFIG.md)

5. **运行程序**
```bash
streamlit run app.py
```

6. **访问界面**

程序启动后，在浏览器中打开 `http://localhost:8501`

## 功能模块说明

### 1. 内置工具

| 工具名称 | 类名 | 功能描述 | 服务状态 |
|---------|------|---------|---------|
| 文件处理 | FileTool | PDF文本/表格提取、PDF转Word | 本地处理 |
| 数据分析 | DataTool | 数据读取、统计计算、图表生成 | 本地处理 |
| 代码运行 | CodeTool | 安全沙箱运行Python代码 | 本地处理 |
| 文献查询 | PaperTool | 学术文献搜索 | **接入真实API** |
| 日程管理 | ScheduleTool | 日程增删查改 | 本地SQLite |
| 文本翻译 | TranslateTool | 多语言文本翻译 | **接入真实API** |
| 文本摘要 | SummaryTool | 文本摘要、关键词提取 | **支持LLM增强** |
| 知识问答 | QATool | 知识查询 | **支持LLM增强** |

### 2. API服务

| 服务类型 | 支持的提供商 | 说明 |
|---------|------------|------|
| 大模型 | OpenAI, 通义千问, 智谱AI, Ollama | 用于意图识别、问答、摘要 |
| 翻译 | 百度翻译, 有道翻译, LLM翻译 | 多语言翻译服务 |
| 文献搜索 | Semantic Scholar, arXiv, CrossRef | 学术文献搜索 |
